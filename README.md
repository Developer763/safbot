# Telegram Moderator Bot

## Установка
1. Установи зависимости:
```bash
pip install -r requirements.txt
```
2. Запусти локально:
```bash
export BOT_TOKEN=ТВОЙ_ТОКЕН
python main.py
```

## Деплой на Railway
- Добавь переменную окружения `BOT_TOKEN` в settings Railway.
- Railway автоматически использует Procfile.
